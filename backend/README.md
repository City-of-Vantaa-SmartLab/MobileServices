## Installation

```bash
$ npm install
```

## Database setup
Database is run in docker for local development.

```bash
# For starting the docker database
npm run db:start

# For stopping the docker database
npm run db:stop
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
